# 專案概覽

## 定位

**PARTI** — 一部建築圖鑑

不是百科全書（對比 Wikipedia），也不是聊天機器人（對比 ChatGPT）。
核心體驗：**「用幾何原理本身、呈現建築知識的結構」**

## 目標用戶

- 建築系在學生、研究生、備課教授（主要）
- 對建築有興趣但知識不多的人（次要）
- 設計圈、Awwwards 評審、跨界讀者（曝光目標）

## 核心體驗

1. 打開首頁 → 看到 4 大入口 + force graph 縮影
2. 選擇分類（人物 / 建築 / 運動 / 地圖）→ 進入該分類總覽
3. 探索（馬賽克 / 年表 / 流動 / 地圖 zoom）→ 點擊感興趣的條目
4. 進入條目頁 → 閱讀內文 + 右側相關詞條 list
5. 任一可點擊文字 → 絲滑跳轉下一頁
6. Cmd+K 隨時跳到任何條目

## 差異化 · v3.1

- **策展權威**：每位建築師、每個流派的呈現由 Kai 個人 POV 決定
- **Glass Editorial 視覺**：4 層 glass + Inter Tight 200 + PingFang TC + 完全無靜態裝飾
- **Hidden-until-Touched UX**：預設冷靜到無聊、互動才爆色、滑鼠停 3 秒淡出
- **N→幾何映射**：1 同心圓 / 2 中垂線 / 3 三點定圓 / 4-6 Delaunay / 7-12 Voronoi / 13+ Force Graph、根據人數自動切換幾何
- **跨流派時期分段**：每位建築師可有 N 個流派標籤 + 各自活躍時期
- **多視角同一資料**：同一份 entity 資料以馬賽克、年表、流動、地圖 4 種視角呈現
- **Scroll-Driven Pyramid Hero**：Landing 「築/圖」字之間三角形 → 像素金字塔（可逆 scroll 動畫）
- **Compare 比較工具**：任選 N 位建築師、按 N→幾何映射自動視覺化（hero piece 王牌互動）
- **AI 生成圖**：全程 AI 生圖、零版權問題、51 P1 slot
