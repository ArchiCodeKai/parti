## 主入口（首頁 4 大）

1. **人物（Architects）** → 馬賽克牆總覽
2. **建築（Buildings）** → 分區年表 + 馬賽克混合
3. **運動（Movements）** → 流動敘事（Apple-style scroll）
4. **地圖（Map）** → 世界地圖 + zoom（標流派發起地 → zoom 顯示建築 / 人物）

## 篩選器（所有頁面側欄）

1. **時代 / Era**：1900s / 1910s … / 2020s 切片
2. **學派 / 學校**：包浩斯、AA、Bartlett、Cooper Union、東海建築系 等
3. **事務所 / 工作室**：OMA、H&dM、SANAA、Foster+Partners、SOM、BIG 等

## 內容附加類型（v1 全部納入）

- 學派 / 學校
- 事務所 / 工作室
- 著作 / 書籍
- 展覽
- 獎項
- 競圖案
- 師生關係（人物之間的隱性傳承線）
- 媒體 / 雜誌

## 不包含（不做）

- 用戶編輯詞條（策展權保留）
- AI 聊天機器人（會稀釋 POV）
- 後端資料庫（純前端 static data）
- 帳號 / 登入（沒必要）
- 留言 / 評論（v3 才考慮、且需人工審核）
- 建築構造 / 材料 / 音場（與策展氣質不合）
- 平面 / 剖面 CAD 重繪（版權問題、工作量爆）

## 入門引導

首頁 hero 下方：**「隨機推送 10 條 500 字以上的精選詞條」**
- 每次 refresh 不同 10 條
- 可能是流派 / 建築師 / 建築物 / 著作
- 點擊直接進入該詞條
