## 分區年表細節

### 分區（v3.1 · 10 年切片 + 密度點）

按 **10 年切片**（更細緻）：
- 1850s / 1860s / 1870s / ... / 2010s / 2020s

每個年代 section 左側有「**密度點視覺**」 — 用密度點群（dot cluster）視覺化該年代的建築量：

```
1920s ● ● ● ● ● ● ●            ← 7 件建築、密度高
1930s ● ● ●                    ← 3 件、密度低
1940s ●                        ← 1 件（二戰）
1950s ● ● ● ● ● ● ● ● ● ●      ← 10 件、戰後爆量
1990s ● ● ● ● ● ●              ← 6 件、解構主義
```

每點 4×4px、灰色、聚成群、hover 點群整體變朱紅。

每區一個 section、section title 大字置中、下方卡片網格排列。

### 時間軸 thin bar

頁面左側永遠有條垂直細軸線、標出每 25 年的點、scroll 跟著移動，告訴用戶「現在在 1950–1975 區」。

```
  │
  │  1850
  │
  │
  │  1900
  │ ●     ← 用戶當前在這
  │
  │  1950
  │
  │  ↓ scroll
```

## 互動

- Hover：邊框朱紅、scale 1.02、鼠標變放大鏡
- 點擊：跳建築物頁（含設計過程、年代事件）

## 建築物頁特色（vs 人物頁）

建築物頁有獨家欄位：
- 設計年 / 完工年 / 拆除年（如有）
- 過程事件（年代軸）：得獎、改建、工安、UNESCO 列入等
- 設計者 + 共同設計者 + 業主
- 流派標籤

## 篩選器

- 時代（25 年切片）
- 流派
- 國家
- 建築師（搜尋）
- 類型：宗教 / 住宅 / 公共 / 商業 / 文化 / 教育 / 紀念性

## 手機版

- 切換按鈕仍保留
- 分區視圖：每區改 1 欄 vertical
- 馬賽克：1 列堆疊
