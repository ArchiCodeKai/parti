# 運動總覽（流動敘事）設計

## 路由

`/movements`

## 設計目的

20 個流派、每個一篇故事。
用 Apple-style scroll、像看電影一樣依序進入每個流派。

## 頁面結構

```
═══════════════════════════════════════════
[Hero  100vh]

           二十個流派

       1850 → 2025 的建築譜系

         scroll ↓
═══════════════════════════════════════════
[流派 1  100vh]

  01 · 新藝術運動
     Art Nouveau · 1890–1910

     代表：Gaudí、Horta
     地點：巴塞隆納、布魯塞爾

     [大型幾何 motif 動畫]
     [代表建築 / 線稿輪播]

     一段 200 字介紹...

     → 進入 Art Nouveau 流派頁
═══════════════════════════════════════════
[流派 2  100vh]
  ...
═══════════════════════════════════════════
[結尾]
  20 / 20

  繼續探索 [/architects] [/buildings]
═══════════════════════════════════════════
```

## 每屏一個流派

20 屏 × 100vh = 整頁 scroll 約 2000vh。

每屏內容：
- 編號（01 / 20）
- 流派中英名
- 年代範圍
- 代表人物（前 3 位）
- 起源地
- 大型幾何 motif（每派一個獨家設計）
- 代表建築線稿輪播
- 200 字 hook 介紹（不是完整介紹、勾起興趣）
- 「進入流派頁」CTA

## Scroll-Driven Animation

每屏進入 viewport 時：
1. 編號 fade in
2. 流派名 stagger reveal
3. 幾何 motif path drawing
4. 文字 fade in
5. 建築線稿輪播啟動

退出 viewport 時：
1. 整屏微微上推
2. 編號顏色淡化

## N→幾何映射（v3.1 整合）⭐

每個流派依「核心人物數量 N」自動套對應幾何視覺：

| N | 流派幾何 | 範例流派 |
|---|---|---|
| **1** | 同心圓 | 自成一派的孤立大師（Kahn 個人主義派、Scarpa）|
| **2** | 中垂線 | 對立 / 對話的兩大代表（如 Smithsons 對 Le Corbusier 晚期）|
| **3** | 三點定圓 | Bauhaus（Gropius / Breuer / Mies 三點定圓）|
| **4–6** | Delaunay 三角網 | International Style、Brutalism、Metabolism |
| **7–12** | Voronoi 細胞 | Deconstructivism、Postmodernism、High-tech |
| **13+** | Force Graph | Modernism（影響上百人的大流派）|

進入流派詳細頁時、自動依該派 `coreArchitects.length` 套對應幾何 hero。

## 每派的代表色

每流派分配一個視覺色（subtle、不是主色）：

| 流派 | 視覺色 |
|---|---|
| Art Nouveau | 暗綠 |
| 包浩斯 | 純黑 |
| 構成主義 | 朱紅（主色） |
| 國際樣式 | 純白 + 黑 |
| 粗獷主義 | 灰青（主色） |
| 代謝派 | 銀灰 |
| 後現代 | 米黃 |
| 解構主義 | 鋼藍 |
| 極少主義 | 純白 |
| ... | ... |

每屏背景以該流派色微微 tint（5% 不透明度）、不破壞主米白底。

## 流派詳細頁（`/movements/[slug]`）

點「進入流派頁」後：
- 完整 800 字介紹
- 該派所有建築師 list
- 該派所有代表建築 list
- 影響其他流派的「傳承樹」
- 反對 / 對立流派
- 右側相關詞條 list
