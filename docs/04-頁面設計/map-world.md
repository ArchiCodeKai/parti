# 地圖總覽（世界地圖 zoom）設計

## 路由

`/map`

## 設計目的

地理空間敘事 — 建築流派發生在哪裡、建築物在哪裡、建築師在哪裡執業。

## 縮放層級（Zoom Levels）

| Zoom | 顯示內容 |
|---|---|
| **L0（世界）** | 20 個流派的「發起地」標記（大圓點 + 流派名） |
| **L1（大洲）** | 該洲所有重要建築、建築師活躍城市 |
| **L2（國家）** | 國家內所有條目分散 |
| **L3（城市）** | 該城市的建築物 pin（細節） |

## L0：世界地圖視圖

```
           ●  ← 代謝派（東京、1960s）
              ●  ← 包浩斯（Dessau、1919）
       ●  ← 構成主義（Moscow、1917）
   ●  ← 加泰隆尼亞現代主義（Barcelona、1890s）
  ●  ← 新藝術（Brussels）
                          ●  ← 批判性地域主義（多地）
       ●  ← 後現代（Las Vegas / 普林斯頓）

  [地圖底色：米白]
  [國境線：細黑、不填色]
  [標記：朱紅 16px 圓點]
```

## 互動

- **Hover 流派標記**：浮現該流派名 + 年代 + 代表 3 位人物
- **點擊流派標記**：zoom in 到該地、進入 L1
- **點擊空白**：zoom out

## 點擊 zoom 動畫

500ms 絲滑 zoom（Mapbox `flyTo`）：
- 地圖移到目標座標
- zoom level 變化
- 標記隨距離出現 / 消失

## L1–L3 標記類型

| 類型 | 圖示 |
|---|---|
| 流派發起地 | 朱紅圓點 + 流派名 |
| 建築物 | 黑色小方形 + 建築名 |
| 建築師活躍城市 | 灰青三角 + 城市名 |
| 學派 / 學校 | 黑色方框 + 校名 |

點擊任一標記 → 跳到該條目頁。

## 篩選器

地圖左上 floating panel：
- 時代切換（1850–現在 slider）
- 流派 toggle
- 顯示類型 toggle（流派 / 建築 / 人物 / 學派）

## 技術選擇

- **Mapbox GL JS**（不用 Google Maps、視覺自由度高）
- **Custom Style**：移除所有商業標記、街道名、自然色彩
- **底色**：米白 #F4EFE6
- **國境線**：黑色 0.5px
- **海洋**：純白 #FFFFFF

## 不要做的

- ✘ 顯示道路 / 城市標籤（噪音）
- ✘ 預設 satellite 衛星圖
- ✘ 3D 地形
- ✘ 預設 Mapbox attribution（需自訂位置）

## 手機版

- 預設 L0 世界視圖
- pinch zoom 切換 level
- 篩選 panel 改 bottom sheet
- 點擊標記 → 直接進詳情頁（不顯示 hover preview）

## 配色一致性

地圖底層保持與全站視覺一致：
- 米白底
- 朱紅 + 灰青標記
- 不破壞 PARTI 整體色感
