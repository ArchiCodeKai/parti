## Goal

生成 PARTI 比較工具頁、**用戶任選 N 位建築師、視覺化關係**、桌機優先。

**v3.1 移除 3 人限制**、依 N 自動套對應幾何：

| N | 幾何 |
|---|---|
| 1 | 同心圓 Concentric |
| 2 | 中垂線 Bisector |
| 3 | 三點定圓 Circumcircle |
| 4-6 | Delaunay 三角網 |
| 7-12 | Voronoi 細胞 |
| 13+ | Force Graph |

要的氣質：**像實驗室分析工具的精準感 + 像建築學論文圖表的論述感 + Glass Editorial**。

## 圓圈視覺規範（必須精確）

圓的大小、彩度、透明度由「離散度」決定：

| 離散度 | 圓大小 | HSL 彩度 | 透明度 |
|---|---|---|---|
| 0.0（最近、風格相近） | 80px | hsl(5, 80%, 50%) 深朱紅 | 1.0 |
| 0.3 | 140px | hsl(5, 65%, 50%) | 0.85 |
| 0.5 | 180px | hsl(5, 55%, 50%) 中朱紅 | 0.75 |
| 0.7 | 220px | hsl(5, 45%, 50%) | 0.65 |
| 1.0（最遠、幾乎無共同點） | 280px | hsl(5, 30%, 50%) 灰紅 | 0.5 |

### 公式

```typescript
size = 80 + dispersion * 200          // 80 → 280px
saturation = 80 - dispersion * 50     // 80% → 30%
opacity = 1.0 - dispersion * 0.5      // 1.0 → 0.5
```

### 視覺意義

> **圓越小 + 越濃 = 越親近**
> **圓越大 + 越淡 = 越疏遠**

像水波紋從中心擴散、近的清晰、遠的朦朧。

## 動畫劇本（給 Claude Design 參考）

### 加入第 2 人（狀態 1 → 狀態 2）

```
1. 第 1 人 timeline fade out
2. 兩個 ● 從卡片飛到視覺區
3. 中垂線從中心向兩側畫出（800ms）
4. 左右特徵 list stagger fade in
```

### 加入第 3 人（狀態 2 → 狀態 3） ⭐

```
1. 中垂線 fade out
2. 對立特徵 fade out
3. 第 3 個 ● fly in 到第三點位置
4. 三點之間連線依序畫出（每線 400ms）
5. 外接圓計算完成、path drawing 1200ms
6. 圓 size/彩度/透明度 同時漸變到目標值
7. 圓內中心點 fade in
8. 共同點 + 差異維度 list fade in
```
